# Foobar2000-UI-Scripts

## 列表: Playlist.js

#### 使用方法:
* 将下列代码粘贴至 Wsh Panel Mod 面板编辑器中:
```
// ==PREPROCESSOR==
// @feature "v1.4"
// @feature "watch-metadb"
// @feature "dragdrop"
// @import "\path\to\common3.js"
// @import "\path\to\Playlist.js"
// ==/PREPROCESSOR==
```
#### 功能：
除了基础的列表操作外还包括或应该包括：
* 歌曲评级
* 封面显示
* 歌曲分组显示
* 等等
